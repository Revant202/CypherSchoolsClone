
## Deployment

To deploy this project run

```bash
  git clone https://github.com/Revant202/CypherSchoolsClone.git
```
starting frontend -
```bash
  cd frontend
```
```bash
  npm start
```
starting API -
first,create .env file in backend folder and add your own MONGODB key
```bash
  cd backend
```
```bash
  npm start
```
